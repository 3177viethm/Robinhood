# Crypto and Equity Instruments News

This endpoint returns list of articles related to an equity instrument by symbol or a currency by ID.

**URL** : `/news/`

**Method** : `GET`

**Auth required** : NO

**Permissions required** : None

**Query parameters**

    - `page` - Optional cursor (integer)
    - `currency_id` - UUID (optional)
    - `symbol` - Ticker symbol (optional)

You must provide either a `symbol` or `currency_id`.

**Request samples**

Essentially the same as `https://midlands.robinhood.com/news/MSFT/`:

```
curl -v https://midlands.robinhood.com/news/?symbol=MSFT/ \
   -H "Accept: application/json"
```

Gather news related to a cryptocurrency (this time, let's try BTC):

```
curl -v https://midlands.robinhood.com/news/?currency_id=d674efea-e623-4396-9026-39574b92b093 \
   -H "Accept: application/json"
```

**Condition** : Sucessfully gather data.

**Code** : `200 OK`
